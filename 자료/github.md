# github 명령어

## 원격지로 올리기
순서 > git init 
      git remote add <name> <url>
      git add .
      git commit -m <comment>
      git push (git push --set-upstream origin master)

* git init
```
새로운 local repository 생성
```
* git add
```
변경된 파일을 storage에 추가
```
* git commit
```
add한 파일을 local repository에 저장
```
* git push
```
local repository를 remote repository에 업로드
```
